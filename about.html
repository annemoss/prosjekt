Questions

1. Servers await requests. The type of request corresponds to the type of server. Web servers respond with web pages etc.

2. Clients send requests, hoping to receive answers.

3. An IP address is a number that other computers use to identify and communicate with the computer.

4. DNS is essentially the same as IP, but easier to memorize. It converts names of websites into IP addresses for us, so we don't have to.

5. The internet is the massive mesh-network that allows all computers connected to it to communicate with eachother. WWW is "just" a way to communicate through HTTP over the internet.

6. HTML is essentially just templating. It helps your browser put things in the correct orders and sizes.
